Система оценки рисков на железнодорожных платформах Лондона

Корпоративная платформа для инспекторов, позволяющая проводить проверки безопасности, сохранять отчёты офлайн и синхронизировать их в единую базу

Vue.js
Laravel
PostgreSQL
Swift (iOS)

Как оценить безопасность сотен платформ в режиме реального времени?

Лондонская железная дорога управляет десятками станций, где необходимо регулярно проводить оценку рисков:

  • Проверка состояния платформ и инфраструктуры;
  • Анализ поведения пассажиров и персонала;
  • Ведение протоколов и отчётности по стандартам PTI (Platform Train Interface).

Ранее эти проверки велись вручную — с бумажными анкетами, а обработка отчётов занимала недели.

Единая база для контроля безопасности

Мы создали веб-панель для администраторов, где можно управлять отчетами платформам станций LNER (Лондонской железной дороги).

Редактор вопросов с системой баллов

Веб-интерфейс позволяет создавать анкеты с разными секциями и вопросами, назначая вес каждого ответа.

Результаты автоматически рассчитывают уровень риска.

Проверки на месте — даже без сети

Инспекторы проходят анкету на платформе, сохраняют результаты офлайн, а система автоматически фиксирует геометку.

После восстановления соединения отчёты синхронизируются с сервером

Просто. Понятно. Функционально.

Веб-панель содержит всё необходимое для управления процессом:
  • Пользователи — создание ролей и управление доступами;
  • Станции и платформы — структурированная база объектов;
  • Разделы и вопросы — конструктор анкет с баллами и зависимыми пунктами;
  • Отчёты — просмотр и экспорт данных в PDF.

От опроса до PDF-отчёта за секунды

После синхронизации с мобильного устройства отчёт формируется автоматически в едином формате. PDF-шаблон содержит:
  • данные о станции и платформе;
  • список секций (General, Public, Employee, Environmental, Equipment);
  • ответы, комментарии и итоговый уровень риска.

Корпоративное iOS-приложение

  • Доступно только для сотрудников через MDM;
  • Работает офлайн (результаты сохраняются локально);
  • Фиксирует GPS-метку при прохождении;
  • Сохраняет анкету после каждого ответа;
  • Автоматически синхронизирует данные при подключении к сети. Скрины: выбор станции → экран вопроса → завершённый отчёт.

Надёжная архитектура корпоративного уровня

  • Frontend: Vue.js SPA;
  • Backend: Laravel (REST API, RBAC, валидации, экспорт PDF);
  • DB: PostgreSQL;
  • Mobile: Swift (iOS), офлайн-режим, фоновая синхронизация;
  • Инфраструктура: защищённый корпоративный хостинг, HTTPS, CI/CD.

Решённые задачи

  • Офлайн-хранение анкет с постепенной синхронизацией;
  • Автоматическая генерация PDF-отчётов с корпоративным шаблоном;
  • Привязка к геолокации и проверка координат;
  • Гибкая настройка анкет под разные станции;
  • Безопасное распределение доступа к данным.
